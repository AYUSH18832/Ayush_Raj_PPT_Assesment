# Task 23: Azure Resource Manager & Cost Management (Easy)

## Objective
Understand resource organization using Resource Groups and monitor costs.

## Environment
- Azure Resource Manager (ARM)
- Azure Cost Management

## Simple Definition
Azure Resource Manager helps organize and manage resources, while Cost Management tracks cloud spending.

---

## Implementation Steps

### 1. Create Resource Group
- Go to Azure Portal → Resource Groups
- Click Create
- Provide name and region
- Create group

---

### 2. View Cost Analysis
- Go to Cost Management + Billing
- Open Cost Analysis
- Review usage by resource group

---

## Learning Outcome
- Organizing resources
- Managing infrastructure using ARM
- Monitoring cloud expenses
