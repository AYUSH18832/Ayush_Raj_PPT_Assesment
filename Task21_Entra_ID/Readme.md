# Task 21: Entra ID (Users, Roles, Basic Identity Management)

## Objective
Manage users and assign roles using Azure Entra ID.

## Environment
- Azure Entra ID
- Azure Portal

## Simple Definition
Azure Entra ID is Microsoft’s cloud-based identity and access management service.

---

## Implementation Steps

### 1. Create User
- Go to Azure Portal → Entra ID
- Select Users → New User
- Configure username and password
- Create user

---

### 2. Assign Role
- Go to Subscriptions → Access Control (IAM)
- Click Add Role Assignment
- Select role (e.g., Reader)
- Assign to created user

---

## Learning Outcome
- Understanding identity management
- Role-based access control (RBAC)
- Secure access management
