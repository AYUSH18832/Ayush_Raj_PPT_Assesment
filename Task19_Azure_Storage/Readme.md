# Task 19: Azure Storage Account (Blob, File, Table)

## Objective
Create and explore different Azure storage services including Blob, File, and Table storage.

## Environment
- Azure Portal
- Azure Storage Account

## Simple Definition
Azure Storage Account provides scalable cloud storage for objects, files, tables, and queues.

---

## Implementation Steps

### 1. Create Storage Account
- Go to Azure Portal → Storage Accounts
- Click **Create**
- Select Subscription & Resource Group
- Choose Region
- Performance: Standard
- Redundancy: LRS (for basic setup)
- Create storage account

---

### 2. Explore Storage Services

#### Blob Storage
- Create a container
- Upload sample file

#### File Storage
- Create file share
- Upload sample file

#### Table Storage
- Create table
- Insert sample entity

---

## Learning Outcome
- Understanding object and file storage
- Working with structured and unstructured data
- Exploring Azure multi-storage services
