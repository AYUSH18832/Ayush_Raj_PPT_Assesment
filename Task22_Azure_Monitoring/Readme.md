# Task 22: Azure Monitoring & Diagnostics (Easy)

## Objective
Enable monitoring and view performance metrics of Azure resources.

## Environment
- Azure Monitor
- Virtual Machine or other resource

## Simple Definition
Azure Monitor collects and analyzes performance and log data from Azure resources.

---

## Implementation Steps

### 1. Enable Monitoring
- Go to Azure Portal → Monitor
- Select resource (e.g., VM)
- View Metrics and Logs

---

### 2. Review Metrics
- CPU usage
- Memory usage
- Network activity

---

## Learning Outcome
- Understanding monitoring tools
- Viewing performance metrics
- Basic diagnostics management
