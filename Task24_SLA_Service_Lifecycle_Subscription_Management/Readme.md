# Task 24: SLA, Service Lifecycle & Subscription Management (Easy)

## Objective
Understand Service Level Agreements (SLA), service lifecycle, and subscription management in Azure.

## Environment
- Azure Portal
- Azure Documentation

## Simple Definition
SLA defines service availability commitments, and subscription management controls billing and resource access.

---

## Implementation Steps

### 1. Review SLA
- Visit Azure SLA documentation
- Check uptime guarantees for services (e.g., VM, Storage)

---

### 2. Manage Subscription
- Go to Subscriptions
- View billing details
- Assign roles and manage access

---

## Learning Outcome
- Understanding SLA commitments
- Managing subscriptions
- Planning governance and service lifecycle
